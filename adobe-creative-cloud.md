# Adobe Creative Clud keyboard short-cut keys
Mac key and their symbol
- **⌘** command
- **⇧** shift
- **^** control
- **⌥** option

| SN | Description | Mac Command |
| -- | ----------- | ----------- |
| 1 | Zoom In / Zoom Out | **cmd(⌘)** + **plus(+)**  /  **cmd(⌘)** + **minus(-)** |
| 2 | Selection | v |


### Mac Shortcut Key
| SN | Description | Shortcut Key | Note |
| -- | ----------- | ----------- | --------------- |
| 1 | Exit Full-Screen Mode | **ctrl(^)** + **cmd(⌘)** + **F** |  |
| 2 | Minimize the front window to the Dock | **cmd(⌘)** + **M** |  |
| 2 | Minimize all windows of the front app | **cmd(⌘)** + **cmd(⌘)** + **M** |  |
